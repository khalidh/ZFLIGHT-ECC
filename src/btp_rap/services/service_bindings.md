# Service Bindings

Create these bindings in ADT after activating the service definitions.

| Service Definition | Binding Type | Suggested Binding | Usage |
|---|---|---|---|
| `ZUI_FLIGHT_MANAGE` | OData V4 - UI | `ZUI_FLIGHT_MANAGE_O4` | Fiori Elements carrier/flight app |
| `ZUI_BOOKING_MANAGE` | OData V4 - UI | `ZUI_BOOKING_MANAGE_O4` | Fiori Elements booking/order/invoice app |
| `ZAPI_FLIGHT_BOOKING` | OData V4 - Web API | `ZAPI_FLIGHT_BOOKING_O4` | External integration API |

After publishing, assign the UI services to an IAM app and business catalog.
